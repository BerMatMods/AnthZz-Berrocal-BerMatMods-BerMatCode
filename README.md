## 🎯 Sobre el Proyecto

Este es mi **portfolio personal profesional** desarrollado con HTML, CSS y JavaScript puro. Diseñado con un estilo moderno, elegante y completamente responsivo, presenta mi trabajo como desarrollador y moderador de WhatsApp Mod.

### ✨ Lo que incluye:

- 🎨 Diseño premium con tema oscuro/claro
- 📱 100% responsivo para todos los dispositivos
- ⚡ Animaciones fluidas y efectos visuales
- 🔢 Contador de visitas en tiempo real
- 🎠 Carrusel de imágenes interactivo
- 📊 Barra de progreso de scroll
- 🌙 Modo claro/oscuro intercambiable

## 📥 Instalación

### Opción 1: Clonar Repositorio

```bash
git clone https://github.com/BerMatModZ/portfolio.git
cd portfolio
```

### Opción 2: Descargar ZIP

1. Haz clic en el botón **Code** (verde)
2. Selecciona **Download ZIP**
3. Extrae el archivo en tu carpeta preferida

### Opción 3: GitHub Pages

1. Ve a **Settings** del repositorio
2. Navega a **Pages**
3. Selecciona la rama **main** y guarda
4. Tu sitio estará disponible en `https://tousuario.github.io/portfolio`

## 🎨 Personalización

### Cambiar Colores

Edita las variables CSS en el `<style>`:

```css
:root {
    --accent-gold: #d4af37;        /* Color principal */
    --bg-primary: #0a0a0a;         /* Fondo principal */
    --text-primary: #ffffff;       /* Texto principal */
}
```

### Modificar Información

Busca y edita en el HTML:

```html
<!-- Nombre y Alias -->
<div class="logo">Anth'Zz <span>Berrocal</span></div>

<!-- Redes Sociales -->
<div class="social-links">
    <a href="TU_ENLACE_GITHUB"><i class="fab fa-github"></i></a>
    <a href="TU_ENLACE_INSTAGRAM"><i class="fab fa-instagram"></i></a>
</div>
```

### Configurar Contador de Visitas

```javascript
// En la sección <script>
let viewCount = 650;  // Cambia el valor inicial
setInterval(updateViewCount, 10000);  // Cada 10 segundos
```

### Verificación de Google

```html
<!-- En el <head> -->
<meta name="google-site-verification" content="TU_CODIGO_DE_VERIFICACION" />
```
